# Manifest - UAE Job Hunting Helper

A gamified job application tracker and networking CRM designed for job seekers in the UAE.

## Features

- **Job Tracker**: Kanban-style board to track applications from 'Applied' to 'Offer'.
- **Networking CRM**: Manage contacts, track interactions, and set follow-up reminders.
- **Gamification**: Earn XP, level up, and unlock achievements by staying consistent.
- **UAE Specifics**: Track visit visa expiry and budget in AED.
- **Dashboard**: Overview of your progress, daily tasks, and visa status.

## Tech Stack

- **Frontend**: React, TypeScript, Vite
- **Styling**: Tailwind CSS, Framer Motion
- **State Management**: Zustand
- **Icons**: Lucide React

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Start the development server:
    ```bash
    npm run dev
    ```

3.  Build for production:
    ```bash
    npm run build
    ```
